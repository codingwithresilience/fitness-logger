# 🏋️ Fitness Logger CLI

A simple terminal-based fitness tracking application built in Python.  
This is **Version 1 (text-only)** of a larger project that will evolve into a full fitness analytics system using SQLite and data visualization.

---

## 🚀 Features

- 📝 Add logs with category (gym, mindset, diet, etc.)
- ⏱️ Automatic timestamps for every entry
- 🔍 Search entries by keyword
- 🏷️ Filter logs by category
- 📖 View recent entries from your log file
- 💾 Lightweight file-based storage (no dependencies)

---

## 🧠 Why I Built This

This project is part of a long-term progression:

- **V1:** Text-based CLI logger (current)
- **V2:** SQLite database version with structured queries
- **V3:** Data visualization using matplotlib (progress tracking, volume, streaks)
- **V4:** Online/cloud version for multi-device syncing

The goal is to build a real-world fitness + self-improvement tracking system while improving Python and software engineering skills.
